Total Number Of Git Commands

  1. pwd- file location
  2. ls- name of the files in the folder
  3. ls -a- files name in the folder (hidden files also)
  4. cd (file name) - change file location
  5. cd ..- back to the same file
  6. git add (file name)- add to the change panel
  7. git add .- adding the changes to the staggered panel
